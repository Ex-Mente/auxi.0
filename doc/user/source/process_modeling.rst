Process Modeling
================
The auxi.modeling.process package contains modules, functions and classes for performing process modeling. Specifically, the package contains a module called "materials" for modeling materials.

Material Modeling
-----------------
The auxi.modeling.process sub-package provides you with modules, classes and functions to model process. The process modeling package contains material models.
